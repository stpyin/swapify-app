# swapify-app
Swapify is a skill exchange platform that enables users to learn new skills by sharing the expertise they already have. Through skill matching, structured learning sessions, and a reward-based credit system, the platform creates a collaborative learning ecosystem where knowledge becomes a valuable resource for everyone.

## Contributions
- Product Ideation
- UX Design
- System Flow Design
- Feature Planning
- Application Development

## Key Features
- Skill Matching
- Skill Swapping
- Credit System
- Course Marketplace

## What I Learned
As one of the project initiators, I helped shape the product concept, designed the user experience and system flow, and co-developed the application with a teammate. This project taught me that innovation is not always about creating something new, but about finding better ways for people to create value together.
